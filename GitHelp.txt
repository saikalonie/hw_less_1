Это две команды для регистрации ваших данных в Git, что бы при пуше вашиз изменений он не запрашивал у эти данные постоянно, пишутся один раз после установки самого Git:



git config --global user.name "Your name"

git config --global user.email "your-email@example.com"

Основные команды для работа с Git

1) Это команды для инициализации Git в вашем новом проекте, пишется в один раз в начале после создания проекта:
git init
2) Это команда для привязки вашего проекта с вашим репозиторием, пишется один раз после команды git init.  в конце команды нужно будет вставить вашу ссылку на ваш репозиторий:
git remote add origin Link


3) Это команды для проверки состояния ваших файлов в проекте.
git status
4) Это команда для добавления в буфер ваших новых файлов и изменений в файлах. после команды add через пробел ставите точку для того чтобы добавить все файлы:
git add .
5) Это команда для сохранения ваших данных которые находятся в буфере. Внутри кавычек пишите название вашего комита:
git commit -m "Commit name"
6) git branch на какой ветке я нахожусьпш
6) Это команда для заливки вашего проекта на ваш репозиторий. После команды origin через пробел пишите название вашей ветки, оно отображается в вашем терминале в скобке (master), (main) или (что-то другое):
git push origin master
